nicobako.github.io

nicobako.dev is my personal site that is still a work in progress and, let's be honest, a playground.

Design

I'm trying something new here. All my friends told me about is cool thing called React, and that I just had to use it, but what do they know? Instead, I'm writing this with plain HTML, and a few other libraries. If you're interested in seeing the source code take a look at the GitHub Repo for this site.

Tailwind

I'm using Tailwind as a CSS framework since I like its concept of atomic CSS.

htmx

I'm using htmx for replacing content on pages to simulate an SPA.

hyperscript

Hyperscript plays very well with htmx and adds JavaScript-y functionality directly in HTML!
